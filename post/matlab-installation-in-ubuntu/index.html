MATLAB installation in Ubuntu (Ubuntu 20.04.1 LTS)

Overview

MATLAB is a powerful programming language/software when it comes to matrix computing. Although it is not a open-source software, as a bioinformatician, I still need to use it sometimes for my research. In this blog, I will introduce how to install MATLAB in your local machine with Ubuntu as your OS.

Procedure

Firstly, we have to download a version of MATLAB that is suitable for your work and your hardware. In my case, I downloaded R2020b for my computer. The link for download can be found here.

After getting the files from the MATLAB official website, we could applied the command unzip in Ubuntu terminal.

unzip -X -K matlab_R2020a_glnxa64.zip -d matlab_2020a_installer

Next, we should open the MATLAB installer and type the command below in Ubuntu terminal.

sudo su
sudo ./install

Applying these commands should activate the installer which is supposed to pop out as either graph-user interface or text-based interface.

Noted that you will be asking to provide a username and a license to proceed the installation. The username should be the same as the username in your local computer. Otherwise, it would warn you that the name does not match the one running in your local machine.

To activate the MATLAB program, in my case, I had to execute activate_matlab.sh by

cd /usr/local/MATLAB/R2020b/bin/
bash activate_matlab.sh

Then, the initiating process will ask you to provide essential information like we did for installation again.

Finally, type matlab in your terminal to run your MATLAB.
